About

This is the first iteration of the Defect Elimination. The tool is being developed to help facilitate AMB's overall Defect Elimination program. AMB has a clear policy and defined approach for Defect Elimination across its mining complex.

Benefits include:

  • all stakeholders can navigate and view data with access control
  • user communication coordination
  • audit trails
  • realtime status instead of periodic reports
  • referential integrity of data
  • save unnecessary manual work and avoid human input errors
  • realtime status instead of periodic reports
  • standardised and queryable data storage

The DE Tool development is a Cohesive project. Feedback and tool issues can be submitted here.
